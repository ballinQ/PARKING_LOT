# Project Documents

This folder keeps phase-specific design, planning, and test instructions out of the project root.

## Layout

- `phase1/` - Phase 1 scope, self-test runbook, and sign-off guidance.
- `phase2/` - Phase 2 roadmap and architecture review.
- `phase2_5/` - Phase 2.5 static Toronto Green P/public parking discovery planning, source review, and self-test guidance.

Root `README.md` remains the running handoff log for agents. Update it whenever meaningful docs, code, tests, reports, or folder structure changes.
